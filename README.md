## Server routes

### Todos

- View all todos<br>
    `GET /api/todos`

- View one todo info<br>
    `GET /api/todos/:id`

- Create new todo<br>
    `POST /api/todos`

- Modify a todo<br>
    `PUT /api/todos/:id`<br>
    `PATCH /api/todos/:id`<br>

- Delete a todo<br>
    `DELETE /api/todos/:id`

### Users

- View all users<br>
    `GET /api/users`

- View one user info<br>
    `GET /api/users/:id`

- Create new user<br>
    `POST /api/users`

- Modify a user<br>
    `PUT /api/users/:id`<br>
    `PATCH /api/users/:id`<br>

- Delete a user<br>
    `DELETE /api/users/:id`
